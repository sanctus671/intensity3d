# Timer Sounds

This directory should contain the timer sound file.

## Required File

- `timer.mp3` - Sound that plays when a timer completes

You can use any short audio file (beep, bell, etc.) for the timer completion sound.

### Example Sources for Timer Sounds

- [Freesound.org](https://freesound.org/)
- [Zapsplat](https://www.zapsplat.com/)
- Create your own using audio editing software

Make sure the file is named `timer.mp3` and placed in this directory.
